# Chat-bot-mgr

College Admission Chatbot - Simple Python bot to answer queries about courses, fees, eligibility & admission process

- **Course Info**: List all available courses like BTech, MBA, BCA, BSc
- **Fees Structure**: Get year-wise fees for each course
- **Duration & Eligibility**: Check course duration and eligibility criteria  
- **Admission Help**: Quick answers to common admission queries
- **Easy to Customize**: Update course data in one dictionary
